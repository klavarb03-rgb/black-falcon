# Black Falcon 🦅

**Desktop платформа для обліку та розподілу матеріальних цінностей**

---

## 🎯 Про проєкт

Black Falcon — система обліку МЦ для організацій з розподіленою мережею менеджерів.

**Статус:** 🚧 В розробці (Фаза 1: MVP)

---

## 🏗️ Архітектура

- **Backend:** Node.js + Express + TypeScript + PostgreSQL
- **Frontend:** Electron + React + TypeScript + SQLite
- **AI:** Ollama (LLaMA Vision + Qwen) + Telegram боти
