# 📊 Інструкція: Експорт звітів у Excel та PDF

## Де знайти кнопки Export?

### 1. **Екран Звітів (Reports)**
Кнопки розташовані в правій частині toolbar, поруч з кнопкою "Сформувати":

```
┌─────────────────────────────────────────────────────────────┐
│  [Зведений] [Інвентар] [Операції]  Від[___] До[___]        │
│                                 [Excel] [PDF] [Сформувати]  │
└─────────────────────────────────────────────────────────────┘
```

### 2. **Dashboard (Головна сторінка)**
Кнопки в правому верхньому куті:

```
┌─────────────────────────────────────────────────────────────┐
│  Вітаємо, Роман Бірюлін                      [Excel] [PDF]  │
│  Адміністратор · Огляд системи                              │
└─────────────────────────────────────────────────────────────┘
```

## Як користуватися?

### Експорт в Excel (.xlsx)

1. **Перейдіть** на екран Reports або Dashboard
2. **Натисніть** кнопку з іконкою завантаження та текстом "Excel"
3. **Дочекайтесь** завантаження файлу
4. **Знайдіть** файл у папці Downloads:
   - Ім'я: `mc-report-2026-03-26.xlsx`
   - Формат: Excel Workbook

**Що в файлі:**
- ✅ Список всіх МЦ з інвентарю
- ✅ Статус (Держ./Волонт.)
- ✅ Кількість, одиниця виміру
- ✅ Дата останнього оновлення

### Експорт в PDF (.pdf)

1. **Перейдіть** на екран Reports або Dashboard
2. **Натисніть** кнопку з іконкою документа та текстом "PDF"
3. **Дочекайтесь** завантаження файлу
4. **Знайдіть** файл у папці Downloads:
   - Ім'я: `mc-report-2026-03-26.pdf`
   - Формат: PDF Document

**Що в файлі:**
- ✅ Форматований звіт для друку
- ✅ Таблиця МЦ з описом
- ✅ Заголовок та дата формування

## Коли файли не завантажуються?

### Помилка: "Помилка: не авторизований"
**Причина:** Ви не залогінені в системі  
**Рішення:** Увійдіть під своїм логіном і паролем

### Помилка: "HTTP 500" або інша помилка сервера
**Причина:** Backend не працює або є проблема з базою даних  
**Рішення:** Перевірте, чи запущений backend сервер

### Файл не відкривається
**Причина:** Браузер/програма не підтримує формат  
**Рішення:**
- Excel: відкрийте через Microsoft Excel, LibreOffice або Google Sheets
- PDF: відкрийте через Adobe Reader, Preview (macOS) або браузер
